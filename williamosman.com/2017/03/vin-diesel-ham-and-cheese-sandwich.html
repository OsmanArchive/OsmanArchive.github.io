Vin Diesel Bust
I have tons of ideas which involve slicing a 3D model into dozens of pieces to cut on my laser. This has been a daunting task since interlocking slices requires parts to be notched. I've built a few projects using this technique but all of the notching was been done manually.
Laser cut skateboard
Manual slicing might be best for a project like the skateboard but it's not feasible for a ham and cheese sandwich. Good news is Autodesk has some amazing slicing software for FREE which can run standalone. 
THE SOFTWARE IS AMAZING. It opens the door for so many projects I would never have attempted. None being impossible without the software, but requiring hours of mundane work or custom scripts. Autodesk lets you change all sorts of slicing parameters and will automatically cut interlocking features into the layers. When you want to export, it fits all of the sliced parts onto the material size you specify. Vin Diesel took 40 sheets of 6.5x4.5 inch ham.
Cut list

Protein Puzzle
The cut files also include a separate layer with part numbers. This is almost mandatory if you have a huge stack of ham. Engraving these numbers onto the ham made assembly really easy.
